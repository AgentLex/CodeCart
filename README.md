# CodeCart
CodeCart: A portable, single-file cognitive memory cartridge for AI collaboration. End long-context pollution and save tokens with DSL-driven knowledge graphs. / 专为 AI 协作设计的便携单文件认知记忆卡带，终结上下文污染并极致节省 Token。

🇺🇸 English Version
CodeCart: The Single-File Cognitive Memory Cartridge
CodeCart is a portable, evolvable, and token-efficient knowledge graph stored in a single HTML file. It solves the "Long Context Inflation" and "Memory Pollution" issues in AI collaboration.

🌟 Key Features
Dual-Drive I/O: High-efficiency DSL instructions instead of bulky JSON.

4D Timeline: Temporal playback to revisit historical decision points.

Lens System: Multi-perspective projection (Engineering, Patent, Security, etc.).

Physical Portability: Single-file HTML, zero-dependency, U-disk ready.

🛠 How to Use
Sync: Click "Copy for AI" to get a compressed Markdown context.

Chat: Paste it to AI. AI replies with DSL commands (e.g., +CLAIM).

Execute: Paste DSL back into the console and click "Execute".

🇨🇳 中文说明
CodeCart：单文件认知记忆卡带
CodeCart 是一款便携、可演化且极致节省 Token 的认知图谱容器，封装于单个 HTML 文件中。它旨在解决 AI 协作中常见的“长上下文膨胀”与“记忆污染”痛点。

🌟 核心特性
非对称双擎 I/O：使用极简 DSL 指令取代臃肿的 JSON，节省 90% 输出 Token。

4D 时间轴：状态时空切片，支持回溯任何历史决策现场。

多维透镜：根据任务切换视角（工程、专利、安全等），精准过滤噪声。

物理级便携：单文件零依赖，优盘即插即用，数据主权完全掌控。

🛠 使用步骤
同步：点击“复制给 AI”获取压缩后的 Markdown 上下文。

对话：贴给 AI。AI 会输出 DSL 指令（如 +CLAIM）。

回写：将指令贴回 CodeCart 控制台并点击“执行”。
