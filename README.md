# 🗂️ CodeCart: The Cognitive Memory Cartridge For AI (v1.3)

**English** | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md)

**Stop AI Memory Pollution. Crystallize Logic. Save Tokens.**

CodeCart is a portable, single-file cognitive cartridge for long AI work sessions. It turns messy conversations into structured logic nodes so users can carry stable context across sessions, devices, and model providers.

---

## ✨ What's New in v1.3

- **Easy Mode**: A zero-learning-curve UI for beginners to record findings without writing DSL.
- **Global 9 Languages**: Auto-detects language via system settings (En, Zh, Ja, Es, Pt, De, Id, Th, Vi).
- **100% True Offline**: Zero dependencies, static QR rendering, and works perfectly without internet.
- **Animated Logo**: New SVG rotating wheel animations for a professional hacker aesthetic.

---

## ⚙️ Core Actions

- **Logic Collapse**: Turn long discussions into atomic `+CLAIM` units.
- **Temporal Updates**: Use `>SUPERSEDE(...)` to evolve old logic with newer decisions.
- **Hard Constraints**: Use `+ANCHOR(...)` to lock non-negotiable project boundaries.
- **Context Sync**: Copy active logic into any AI chat or API workflow with one click.
