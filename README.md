# 🗂️ CodeCart: The Cognitive Memory Cartridge For AI
# 🗂️ CodeCart：AI认知记忆卡带

**Stop AI Memory Pollution. Crystallize Logic. Save Tokens.**  
**停止 AI 记忆污染，结晶逻辑，节省 Token。**

CodeCart is a single-file cognitive cartridge for long AI work sessions. It turns messy conversations into structured logic nodes so users can carry stable context across sessions, devices, and model providers.
CodeCart 是一个面向长 AI 工作流的单文件认知卡带工具。它把混乱对话压缩为结构化逻辑节点，帮助用户跨 session、跨设备、跨模型供应商维持稳定上下文。

---

## ✨ Core Features / 核心特性

- **Logic Collapse**: Turn long discussions into atomic `+CLAIM` units.
- **Temporal Updates**: Use `>SUPERSEDE(...)` to replace outdated logic with newer decisions.
- **Hard Constraints**: Use `+ANCHOR(...)` to lock non-negotiable rules.
- **Portable Context**: Copy active logic into any AI chat or API workflow.
- **Single-file Delivery**: No install, no backend, no build process.
