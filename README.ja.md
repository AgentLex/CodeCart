# 🗂️ CodeCart: AIのための認知メモリカートリッジ (v1.3)

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | **日本語**

**AIの記憶汚染を防ぎ、ロジックを結晶化し、トークンを節約する。**

CodeCartは、長時間のAI作業セッション向けに設計された、ポータブルで単一ファイルの認知カートリッジツールです。複雑な会話を構造化されたロジックノードに圧縮し、セッション、デバイス、AIモデルのプロバイダーを越えて安定したコンテキスト（文脈）を維持するのに役立ちます。

---

## ✨ v1.3 の新機能

- **簡易モード (Easy Mode)**: 初心者向けのUI。DSL構文を学ばなくても、発見やルールを簡単に入力・保存できます。
- **全9言語対応**: ブラウザのタイムゾーンから言語を自動検出（英語、簡体/繁体中国語、日本語、スペイン語、ポルトガル語、ドイツ語、インドネシア語、タイ語、ベトナム語）。
- **100% 完全オフライン**: バックエンドやインストールは不要。静的なQRコードレンダリングを内蔵し、インターネット接続なしでも完璧に動作します。
- **アニメーションLogo**: SVGロゴに車輪の回転アニメーションを追加し、よりハッカーらしい外観に。

---

## ⚙️ コア機能とワークフロー

- **ロジックの圧縮 (Collapse)**: 長い議論をアトミック（最小単位）な `+CLAIM` ユニットに変換します。
- **時系列アップデート (Supersede)**: `>SUPERSEDE(...)` を使用して、古いロジックを新しい決定で置き換えます。
- **厳格な制約 (Anchor)**: `+ANCHOR(...)` を使用して、変更不可のプロジェクト境界やルールをロックします。
- **コンテキストの同期**: ワンクリックで有効なロジックをコピーし、任意のAIチャットやAPIワークフローに同期させます。

## 🐣 対象ユーザー

- **プロンプトエンジニア**: 長時間のセッションにおけるコンテキストの肥大化とAIの「記憶喪失」を防ぎたい方。
- **研究者・ライター**: 終わりのないチャット履歴ではなく、バージョン管理されたロジックを必要とする方。
- **開発者**: LLMの外部に、軽量で依存関係のないメモリ管理レイヤーを求めている方。
