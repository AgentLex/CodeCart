# 🗂️ CodeCart：AI 認知記憶卡帶 (v1.3)

[English](README.md) | [简体中文](README.zh-CN.md) | **繁體中文** | [日本語](README.ja.md)

**停止 AI 記憶污染，結晶邏輯，節省 Token。**

CodeCart 是一個面向長 AI 工作流的便攜式、單一檔案認知卡帶工具。它把混亂的對話壓縮為結構化邏輯節點，幫助用戶跨 Session、跨設備、跨模型供應商維持穩定的上下文。

---

## ✨ v1.3 新特性

- **簡易模式 (Easy Mode)**: 新手友好的填空式記錄模式，無需學習 DSL 語法即可上手。
- **全球 9 語言支援**: 根據瀏覽器時區自動檢測語言（中簡/繁、英、日、西、葡、德、印尼、泰、越）。
- **100% 純離線**: 零依賴單一檔案，內建靜態 QR Code 渲染，斷網環境下依然完美運作。
- **動態互動 Logo**: 為 SVG 標誌增加了輪軸旋轉動畫，視覺更具極客風格。

---

## ⚙️ 核心工作流

- **邏輯坍縮 (Collapse)**: 將冗長的討論壓縮為原子級的 `+CLAIM` 單元。
- **時序更新 (Supersede)**: 使用 `>SUPERSEDE(...)` 以新決策演進舊邏輯。
- **硬性約束 (Anchor)**: 使用 `+ANCHOR(...)` 鎖定不可動搖的專案邊界。
- **上下文同步**: 一鍵複製有效邏輯，無縫同步至任何 AI 聊天介面。
